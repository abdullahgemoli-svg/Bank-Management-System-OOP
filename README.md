# 🏦 Bank Management System (OOP)

A comprehensive **Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles.

This project simulates a real-world banking system through a console-based interface, providing secure client management, user authentication, currency exchange, financial transactions, and role-based access control.

# ✨ Features

## 👤 Client Management

- Add new clients
- Update client information
- Delete clients
- Search for clients
- View all clients
- Calculate total balances

---

## 💰 Banking Transactions

- Deposit money
- Withdraw money
- Transfer money between clients
- View transfer history

---

## 🔐 User Management

- User Login
- Add new users
- Update users
- Delete users
- Search users
- Display all users

---

## 🛡 User Permissions

The system supports role-based permissions allowing administrators to control user access to different system modules.

Examples:

- Client Management
- Transactions
- User Management
- Currency Exchange
- Reports

---

## 💱 Currency Exchange

- Display currencies
- Search currencies
- Update exchange rates
- Currency calculator
- Currency conversion

---

## 📁 File-Based Database

The application stores data using text files without requiring an external database.

Files used:

- Clients.txt
- Users.txt
- Currencies.txt
- LoginRegister.txt
- TransferLog.txt

# 🏗 Project Architecture

The project follows Object-Oriented Programming principles and separates responsibilities into multiple classes.

Examples include:

- clsBankClient
- clsUsers
- clsCurrency
- clsDate
- clsUtil
- clsInputValidate
- clsScreen
- clsMainScreen
- clsTransactionsScreen
- clsManageUsersScreen

This architecture makes the project modular, maintainable, and easy to extend.


3. Build the solution.

4. Run the project.
